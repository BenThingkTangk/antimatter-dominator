02 Logo & wordmark

One mark. Two surfaces. Δ is always Unicode.

The canonical wordmark is ΔTOM — Greek capital Delta (U+0394), T, teal O ring, M. Never substitute Latin A. Lockup pairs the orbital icon at left.

Dark · canonical
Light · cream
Icon-only
Use at favicons, app tiles, OG cards. Never below 16px square.
Lettermark
Δ alone. Cabinet Grotesk 800. Use for collapsed nav, social avatars.
Wordmark only
Δ + T + teal O ring + M. Equal Δ height of clear space on all sides.
Hard rules
Do not break the wordmark.
  • The first character must always be Greek capital Delta Δ (U+0394). Never Latin A.
  • Δ, T, M render in light cream #e8e8ea on dark; near-black on cream.
  • The O is a teal ring accent — never the Δ.
  • Glow only. No drop shadows on the logo. Maintain Δ-height clear space.

06 Accessibility

WCAG AA, keyboard-first, motion-aware.

Contrast

Body text on dark canonical reaches 13.5:1. Teal primary on dark reaches 6.2:1 — passes AA Large and UI.

Focus

All interactive elements expose a 4px teal ring via :focus-visible. Skip-link at top of page.

Motion

prefers-reduced-motion: reduce disables loaders, orbital spin, glow bloom, pulse, sweep, waveform, ripple.

Try tabbing through the page — every focusable element has a visible teal ring at 4px against background. The skip-link appears at top-left on first Tab.

07 Implementation

Drop-in install.

1 · Include the CSS

<link rel="stylesheet" href="css/atom-tokens.css" />
<link rel="stylesheet" href="css/atom-animations.css" />
<link rel="stylesheet" href="css/atom-components.css" />

2 · Load fonts

<link href="https://api.fontshare.com/v2/css?f[]=cabinet-grotesk@400,500,700,800&f[]=satoshi@300,400,500,700&display=swap" rel="stylesheet" />
<link href="https://fonts.googleapis.com/css2?family=JetBrains+Mono:wght@300;400;500&display=swap" rel="stylesheet" />

3 · Mount the loader (optional)

<!-- copy markup from components/atom-loader.html into <body> -->
<script src="js/atom-loader.js"></script>
<script>
  // Boot once on page load — default 2500ms
  ATOMLoader.init({ duration: 2500, autoStart: true });

  // Or trigger manually
  ATOMLoader.show();
  ATOMLoader.hide();
  ATOMLoader.play({ duration: 1500 });
</script>

4 · Toggle theme

// On dark (canonical):
document.documentElement.setAttribute('data-theme', 'dark');

// On light cream:
document.documentElement.setAttribute('data-theme', 'light');

5 · Render the wordmark correctly

<!-- Prefer the SVG -->
<img src="assets/atom-wordmark.svg" alt="ΔTOM" />

<!-- Or inline text — Δ is U+0394, never Latin A -->
<span class="atom-wordmark" aria-label="ΔTOM">
  <span>Δ</span><span>T</span><span class="atom-wordmark__o">O</span><span>M</span>
</span>
